# Сервер адаптивной очереди
## Инструкция по запуску
1. Необходимо установить JDK 17 или выше, а также СУБД PostgreSQL.
2. Склонировать репозиторий и перейти в корневую папку проекта
3. Так как в зависимостях проекта используется библиотека `ru.bmstu.naburnm8.adaptiveQueue`, необходимо добавить ее в локальный репозиторий Maven. Для инструкций перейдите по [ссылке](https://github.com/naburnm8/adaptiveQueue)
4. Добавить переменные среды в окружение запуска или в параметры запуска:
   - `DB_URL` - URL для подключения к базе данных. Например, `localhost:5432/mydb`
   - `DB_USER` - имя пользователя для подключения к базе данных. Например, `postgres`
   - `DB_PASS` - пароль для подключения к базе данных. Например, `password`

5. Выполнить команду `./gradlew run` для запуска сервера
6. Сервер будет запущен на порту 8081.

## Документация API

После запуска сервера доступна документация API по адресу `http://localhost:8081/swagger-ui/index.html`. В документации описаны все доступные эндпоинты, их параметры и примеры запросов и ответов.
